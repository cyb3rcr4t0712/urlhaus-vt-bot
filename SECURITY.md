# Security Policy

This project is provided for defensive security research and does not handle sensitive data or authentication.

There are no known security vulnerabilities and no special security review or responsible disclosure process for this code.

If you find a code bug or improvement, please open a regular GitHub issue or pull request.

## Acceptable Use

✅ Allowed:
- Using this bot with your own API keys for threat intel
- Contributing improvements, bug fixes, or new features
- Using this as a reference for security automation

❌ Not allowed:
- Using this tool to distribute, host, or execute malware
- Downloading or executing anything from malicious URLs processed by this bot
- Misrepresenting this tool as anything except defensive research software
